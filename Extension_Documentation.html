QR Code Generator

Chrome Extension

Description
This Chrome extension is designed to generate QR codes for any URL or Word. And we can download this QR codes as an image. Which we can then use for different purposes.

To see how this QR Code Generator Extension work you need to follow the following steps.

Step 1
Click on this icon to open QR Code Generator Extension.

Step 2
Paste an URL or Word in given input field of QR Code Generator Extension.

Step 3
Click on Generate QR Code button to generate QR code.

Step 4
Click on Download button to download generated QR code as an Image.

Step 5
Click on this icon to close QR Code Generator Extension.

Click on the download button to download project file of QR Code Generator Extension. Download
